# Тестовое здание VK

> В качестве State Manager'a был выбран Tanstack Query, потому что основная работа идет с асинхронными данными и нет других глобальных состояний.

Установка зависимостей

```bash
npm i
```

Старт mock API

```bash
npm run server
```

Старт dev server

```bash
npm run dev
```
